# 📝 Smart Todo App

A modern and interactive Todo List web application built with **JavaScript**, featuring task progress tracking, local storage persistence, and a clean user experience.

---

## 🚀 Features

* ✅ Add, edit, and delete tasks
* 🔄 Mark tasks as **In Progress**
* ✔️ Mark tasks as **Completed**
* 🌙 Dark mode toggle
* 💾 Save tasks using Local Storage
* 🔔 Alerts & sound feedback
* 📊 Task counters (Total & Completed)
* 🔝 Scroll to top button

---

## 🎯 Project Purpose

This project was built to practice:

* DOM manipulation
* State management using JavaScript
* Local Storage handling
* UI/UX improvements without frameworks
